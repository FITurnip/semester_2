A

<a>...</a>
Defines a hyperlink

<abbr>...</abbr>
Defines an abbreviation or an acronym

<address>...</adress>
Defines contact information for the author/owner of a document

<applet>...</applet>
Defines an embedded applet(Plug-in) in HTML 4

<area>...</area>
Defines an area inside an image map (an image map is an image with clickable areas)

<article>...</article>
Defines independent, self-contained content

<aside>...</aside>
Defines some content aside from the content it is placed in

<audio>...</audio>
Defines an embedded sound content in a document

B

<b>...</b>
Defines bold text

<base />
Defines the base URL and/or target for all relative URLs in a document

<basefont />
Defines a default text-color, font-size or font-family for all the text in an HTML document in HTML 4

<bdi>...</bdi>
Defines an isolated part of text that might be formatted in a different direction from other text outside it

<bdo>...</bdo>
Defines the current text direction is overrided

<big>...</big>
Defines bigger text.

<blockquote>...</blockquote>
Defines a section that is quoted from another source

<body>...</body>
Defines the document's body

<br />
Defines a single line break

<button>...</button>
Defines a clickable button

C

<canvas>...</canvas>
Defines a drawing graphics, on the fly, via scripting

<caption>...</caption>
Defines a table caption

<center>...</center>
Defines a center-align text

<cite>...</cite>
Defines the title of a creative work

<code>...</code>
Defines the title of a creative work

<col>...</col>
Defines column properties for each column within a <colgroup> element

<colgroup>...</colgroup>
Defines a group of one or more columns in a table for formatting

D

<data>...</data>
Defines a machine-readable translation of a given content

<datalist>...</datalist>
Defines a list of pre-defined options for an <input> element

<dd>...</dd>
Defines a term/name in a description list

<del>...</del>
Defines text that has been deleted from a document

<details>...</details>
Defines additional details that the user can open and close on demand

<dfn>...</dfn>
Defines a term that is going to be defined within the content

<dialog>...</dialog>
Defines a dialog box or subwindow

<dir>...</dir>
Defines the ist directory titles in HTML 4

<div>...</div>
Defines a division or a section in an HTML document

<dl>...</dl>
Defines a description list

<dt>...</dt>
Defines a term/name in a description list

E

<em>...</em>
Defines emphasized text

<embed>...</embed>
Defines a container for an external resource, such as a web page, a picture, a media player, or a plug-in application

F

<fieldset>...</fieldset>
Defines group related elements in a form

<figcaption>...</figcaption>
Defines a caption for a <figure> element

<figure>...</figure>
Defines self-contained content, like illustrations, diagrams, photos, code listings, etc

<font/>
Defines the font face, font size, and color of text in HTML 4

<footer>...</footer>
Defines a footer for a document or section

<form>...</form>
Defines an HTML form for user input.

<frame>...</frame>
Defines one particular window (frame) within a >frameset< in HTML 4

<frameset>...</frameset>
Defines a frameset in HTML 4

H

<h1> to </h6>
Defines HTML headings

<head>...</head>
Defines a container for metadata (data about data) and is placed between the <html> tag and the <body> tag

<header>...</header>
Defines a container for introductory content or a set of navigational links

<hr>...</hr>
Defines a thematic break in an HTML page (e.g. a shift of topic)

<html>...</html>
Defines the root of an HTML document and he container for all other HTML elements (except for the <!DOCTYPE> tag)

I

<i>...</i>
Defines a part of text in an alternate voice or mood

<iframe>...</iframe>
Defines an inline frame

<img/>
Defines an image in an HTML page

<input>...</input>
Defines an input field where the user can enter data

<ins>...</ins>
Defines a text that has been inserted into a document

K

<kbd>...</kbd>
Defines keyboard input

L

<label>...</label>
Defines a label for several elements

<legend>...</legend>
Defines a caption for the <fieldset> element

<li>...</li>
Defines a list item

<link>...</link>
Defines the relationship between the current document and an external resource.

M

<main>...</main>
Defines the main content of a document

<map>...</map>
Defines an image map

<mark>...</mark>
Defines text that should be marked or highlighted

<meta />
Defines metadata about an HTML document

<meter>...</meter>
Defines a scalar measurement within a known range, or a fractional value

N

<nav>...</nav>
Defines a set of navigation links

<noframe>...</noframe>
Defines its act as a fallback tag for browsers that did not support frames in HTML 4

<noscript>...</noscript>
Defines an alternate content to be displayed to users that have disabled scripts in their browser or have a browser that doesn't support script

O

<object>...</object>
Defines a container for an external resource

<ol>...</ol>
Defines an ordered list

<optgroup>...</optgroup>
Defines roup related options in a <select> element (drop-down list)

<option>...</option>
Defines an option in a select list

<output>...</output>
Defines represent the result of a calculation (like one performed by a script)

P

<p>...</p>
Defines a paragraph

<param />
Defines arameters for an <object> element

<picture>...</picture>
Defines flexibility in specifying image resources

<pre>...</pre>
Defines preformatted text

<progress>...</progress>
Defines the represents the completion progress of a task

Q

<q>...</q>
Defines a short quotation

R

<rp>...</rp>
Defines parentheses around a ruby text, to be shown by browsers that do not support ruby annotations

<rt>...</rt>
Defines an explanation or pronunciation of characters (for East Asian typography) in a ruby annotation

<ruby>...</ruby>
Defines a ruby annotation

S

<s>...</s>
Defines text that is no longer correct, accurate or relevant

<samp>...</samp>
Defines sample output from a computer program

<script>...</script>
Defines a embedded client-side script (JavaScript)

<section>...</section>
Defines a section in a document

<select>...</select>
Defines a drop-down list

<small>...</small>
Defines smaller text

<source />
Defines multiple media resources for media elements, such as <video>>, <audio>, and <picture>.

<span>...</span>
Defines an inline container used to mark up a part of a text, or a part of a document

<strike>...</strike>
Defines define strikethrough text in HTML 4

<strong>...</strong>
Defines text with strong importance

<style>...</style>
Defines style information (CSS) for a document

<sub>...</sub>
Defines subscript text

<summary>...</summary>
Defines a visible heading for the <details> element

<sup>...</sup>
Defines superscript text

<svg>...</svg>
Defines a container for SVG graphics

T

<table>...</table>
Defines an HTML table

<tbody>...</tbody>
Defines group the body content in an HTML table

<td>...</td>
Defines a standard data cell in an HTML table

<template>...</template>
Defines a container that hold some HTML content hidden from the user when the page loads

<textarea>...</textarea>
Defines a multi-line text input control

<tfoot>...</tfoot>
Defines group footer content in an HTML table

<th>...</th>
Defines a header cell in an HTML table

<thead>...</thead>
Defines group header content in an HTML table

<time>...</time>
Defines a specific time (or datetime)

<title>...</title>
Defines the title of the document

<tr>...</tr>
Defines a row in an HTML table

<track>
Defines text tracks for <audio> or <video> elements

<tt>...</tt>
Defines teletype text in HTML 4

U

<u>...</u>
Defines some text that is unarticulated and styled differently from normal text

<ul>...</ul>
Defines an unordered (bulleted) list

V

<var>...</var>
Defines a variable in programming or in a mathematical expression

<video>...</video>
Defines embed video content in a document, such as a movie clip or other video streams

W

<wbr>...</wbr>
Defines a line-break where in a text it would be ok to add
